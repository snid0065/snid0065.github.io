Week 3 Tags

Text and Semantics Tags

h1-6:
Used to assign headings, weighted largest to smallest by rank
p:
Used to assign paragraph designation to text
q:
Used to show quotation marks around framed text
blockquote:
Used for a large stand alone quote from another source
em:
Used to show emphasis in form of italics
mark:
Used to show emphasis in the form of highlighting framed text
i:
Used to show a technical term, ie. a ship name, a book title, a thought, sarcasm, another language.
sub and sup:
Used to make subscript or super script
hr:
Used for a break in text, appears as a line
em:
Used to show emphasis in form of italics

Week 4 Tags

Image Tags
Tag Description
img: Used to display an image on the page
src: Used to provide the relative or absolute path to the image
alt: Used to describe an image to assisted technologies which can read this alternative text
title: Used to add a title to the image, seen when tooltip is hovered over image
width and height: Used to override height and width dimenstions of an image
