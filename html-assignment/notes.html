Week 3 Tags

Text and Semantics Tags

h1-6:
Used to assign headings, weighted largest to smallest by rank
p:
Used to assign paragraph designation to text
q:
Used to show quotation marks around framed text
blockquote:
Used for a large stand alone quote from another source
em:
Used to show emphasis in form of italics

Image Tags

img:
Used to display an image on the page
src:
Used to provide the relative or absolute path to the image
alt:
Used to describe an image to assisted technologies which can read this alternative text
title:
Used to add a title to the image, seen when tooltip is hovered over image
width and height:
Used to override height and width dimenstions of an image
